# Description
This is a food sorting puzzle game. Players match food items into  correct slots before time runs out. There are four different items and have four similar of it which have to be placed in the slots with similar items. When you win, a happy dog appears with a plate of food as a reward, and if you lose, a crying dog appears as the dog don't receive anything and it's still hungry. The game is all about foucus and speed. click 'r' / 'R' to restart the game.
